LESSON 1

The WORLD WIDE WEB is collection of documents being communicated between computers all over the world. These documents are called HTML’s (hypertext markup language). When you click on a web page, your PC sends a request to the server associated with that page. This request is sent via a protocol called HTTP.


What is HTML?

Text Content - what you see

Mark Up - what it looks like

References to Other Docs - images, videos, etc…

WHITESPACE

In HTML text on multiple lines compresses to one space. To add a break we use a (br TAG). The break is a Void Tag. Every (br) equals one line break. There is no limit to the amount of breaks that you can use.

You can also use a (p TAG) to help group text in a paragraph format...for example (p) contents, contents, centents (/p) groups everything between p Tags together with a line break before and after the contents.
